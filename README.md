# 浜松市 乳児等通園支援事業マップ

浜松市の公開オープンデータを利用し、乳児等通園支援事業の実施施設を地図上に表示する静的Webアプリです。年齢と曜日で表示内容を切り替えられます。

## GitHub Pages
https://suzuking001.github.io/daredemo_map/

## 特徴
- 浜松市オープンデータから施設CSVを取得して表示
- 年齢・曜日・施設種別で表示を切り替え
- 利用枠の有無に応じたマーカー色分け（可=緑 / 不可=赤 / 未選択=灰）
- 施設名ラベルを常時表示
- 現在地表示と2km/5km圏のガイド円
- About モーダルでライセンス情報を表示

## 使い方
1. `index.html` をブラウザで開きます。
2. メニューから年齢・曜日・施設種別を選択します。
3. 地図上のマーカーをクリックすると詳細が表示されます。
